## Git Sync Workflow

**ALWAYS do this at the start of every session:**
- Run `git pull` to get latest changes from other machines

**ALWAYS do this at the end of every session:**
- Run `git add .`
- Run `git commit -m "describe what you worked on"`
- Run `git push`
